관리자의 계정은 id: manager, pw: manager로 설정되어 있습니다.

계정이 없는 경우, 우측 상단의 프로필 버튼을 누르고 회원가입 버튼을 누릅니다. 그리고 id, pw, 닉네임을 입력한 후 가입하기 버튼을 누릅니다.

계정이 있거나 계정을 생성한 경우, 다시 프로필 버튼을 눌러 로그인 버튼을 누릅니다. 자신의 아이디와 비밀번호를 입력하여 로그인을 합니다.

메인 화면에는 랜덤하게 추천해주는 게임이 중앙에 표시되고 아래 게임 카탈로그 버튼을 눌러 프로그램에 저장된 게임 목록을 볼 수 있습니다.

이 목록 중 찾는 게임이 없다면 검색 버튼을 눌러 찾을 수 있고 존재하는 게임이라면 게임 정보 창으로 이동하고 없는 게임이면 존재하지 않는다는 메시지가 표시됩니다.

게임 정보 창을 가면 게임에 대한 기본적인 설명, 게임을 하는데 알면 좋은 게임 팁, 그리고 다른 사용자의 리뷰를 볼 수 있습니다.

로그인이 된 경우에는 리뷰 창에서 자신의 리뷰를 남길 수 있지만 비로그인 상태에서는 다른 사용자의 리뷰만 볼 수 있습니다.

이 프로그램에서 자신이 찾는 게임이 없다면 메인에서 프로필 버튼에서 게임 정보 요청 버튼을 눌러 관리자에게 요청할 수 있습니다.

사용자의 계정으로 요청을 완료했다면 관리자가 이 요청을 처리할 수 있습니다.

관리자의 계정으로 로그인을 한 후, 프로필 버튼 중 요청 목록 버튼을 누르면 사용자가 요청한 게임의 목록을 볼 수 있습니다.

관리자가 수락을 한다면 게임 카탈로그에 게임이 추가되고 거부를 한다면 추가되지 않습니다.
